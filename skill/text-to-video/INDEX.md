## 📝 检查清单

### 制作视频前
- [ ] 已阅读 [SKILL.md](./SKILL.md)
- [ ] 已安装 Node.js 和 FFmpeg
- [ ] 已创建项目并安装依赖
- [ ] 已复制代码模板

### 准备内容时
- [ ] 参考 [内容拆分指南](./references/content-splitting.md)
- [ ] 字幕递进不重复
- [ ] 每个镜头 3-5 秒

### 搜索图片时
- [ ] 参考 [图片搜索指南](./references/image-sourcing.md)
- [ ] 使用免费商用图片
- [ ] 图片至少 1920x1080
- [ ] 下载到 public/media/ 目录

### 渲染前
- [ ] 在 studio 中预览过
- [ ] 检查所有图片正常显示
- [ ] 字幕清晰可读
- [ ] JSON 格式正确

### 遇到问题时
- [ ] 查看 [故障排查指南](./assets/troubleshooting.md)
- [ ] 查看 [FAQ](./assets/FAQ.md)
- [ ] 检查错误日志
