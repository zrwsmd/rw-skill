# 快速使用指南

## 第一次使用（5分钟）

### 1. 创建项目
```bash
npx create-video@latest my-first-video
cd my-first-video
npm install
```

### 2. 复制模板文件
```bash
# 从 skill 目录复制模板
cp E:/claude-project/skill-project/skill/text-to-video/scripts/ShotCard.tsx src/
cp E:/claude-project/skill-project/skill/text-to-video/scripts/index.tsx src/
cp E:/claude-project/skill-project/skill/text-to-video/scripts/data-template.json src/videoData.json
```

### 3. 创建图片目录
```bash
mkdir -p public/media
```

### 4. 下载示例图片
访问 https://www.pexels.com 下载2-3张图片到 `public/media/`，重命名为：
- image1.jpg
- image2.jpg
- image3.jpg

### 5. 编辑配置文件
编辑 `src/videoData.json`：

```json
{
  "segments": [
    {
      "id": "demo",
      "category": "示例视频",
      "source": "原创",
      "shots": [
        {
          "image": "media/image1.jpg",
          "subtitle": "这是第一个镜头的字幕",
          "durationSeconds": 4
        },
        {
          "image": "media/image2.jpg",
          "subtitle": "这是第二个镜头的字幕",
          "durationSeconds": 3
        },
        {
          "image": "media/image3.jpg",
          "subtitle": "这是第三个镜头的字幕",
          "durationSeconds": 3
        }
      ]
    }
  ]
}
```

### 6. 预览视频
```bash
npx remotion studio
```

浏览器会自动打开，可以实时预览视频效果。

### 7. 渲染视频
```bash
npx remotion render src/index.tsx Video out/my-first-video.mp4 --codec=h264
```

完成！视频保存在 `out/my-first-video.mp4`
