# 文字转视频工作流程

```
┌─────────────────────────────────────────────────────────────┐
│                     1. 用户提供文字内容                        │
│                                                             │
│  示例："苹果公司今天发布了新款 iPhone，配备 A17 芯片..."      │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     2. 理解需求和风格                          │
│                                                             │
│  • 视频类型：新闻/教程/产品/故事                              │
│  • 时长要求：30秒/1分钟/2分钟                                 │
│  • 风格偏好：正式/轻松/专业                                   │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     3. 内容拆分                               │
│                                                             │
│  将文字拆分为多个镜头（参考 content-splitting.md）            │
│                                                             │
│  镜头1: "苹果公司今天举行新品发布会"                          │
│  镜头2: "推出搭载 A17 芯片的 iPhone 15 Pro"                  │
│  镜头3: "起售价 999 美元，9月15日开售"                        │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     4. 图片搜索                               │
│                                                             │
│  为每个镜头找匹配图片（参考 image-sourcing.md）               │
│                                                             │
│  镜头1 → Pexels: "apple product launch"                     │
│  镜头2 → Pexels: "iphone closeup"                           │
│  镜头3 → Pexels: "price tag technology"                     │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     5. 创建项目                               │
│                                                             │
│  npx create-video@latest my-video                           │
│  cd my-video                                                │
│  npm install                                                │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     6. 复制模板文件                           │
│                                                             │
│  • 复制 scripts/ShotCard.tsx → src/                         │
│  • 复制 scripts/index.tsx → src/                            │
│  • 复制 scripts/data-template.json → src/videoData.json     │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     7. 下载图片                               │
│                                                             │
│  mkdir -p public/media                                      │
│  下载图片到 public/media/ 目录                               │
│  重命名为有意义的文件名                                       │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     8. 编辑数据文件                           │
│                                                             │
│  编辑 src/videoData.json，填入：                             │
│  • 图片路径                                                  │
│  • 字幕内容                                                  │
│  • 时长（秒）                                                │
│  • 分类和来源                                                │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     9. 预览视频                               │
│                                                             │
│  npx remotion studio                                        │
│  在浏览器中实时预览                                          │
│  调整字幕、时长、样式                                         │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                     10. 渲染视频                              │
│                                                             │
│  npx remotion render src/index.tsx Video out/output.mp4 \   │
│    --codec=h264                                             │
│                                                             │
│  输出：out/output.mp4                                        │
└─────────────────────────────────────────────────────────────┘
```

## 时间估算

| 步骤 | 时间 |
|-----|------|
| 内容拆分 | 5-10分钟 |
| 图片搜索和下载 | 10-20分钟 |
| 项目配置 | 5分钟 |
| 数据文件编辑 | 5分钟 |
| 预览和调整 | 5-10分钟 |
| 渲染视频 | 2-5分钟 |
| **总计** | **30-55分钟** |

## 快速检查清单

创建视频前检查：
- [ ] 所有镜头的字幕是否递进、不重复？
- [ ] 图片是否全部下载到 `public/media/` 目录？
- [ ] `videoData.json` 中的图片路径是否正确？
- [ ] 每个镜头时长是否在 3-5 秒？
- [ ] 总时长是否符合预期？

渲染前检查：
- [ ] 是否已运行 `npm install`？
- [ ] 是否已在 `npx remotion studio` 中预览过？
- [ ] 图片是否都正常显示（无黑屏）？
- [ ] 字幕是否清晰可读？

## 常见问题快速解决

| 问题 | 解决方案 |
|-----|---------|
| 黑屏 | 检查图片路径，确保使用 `staticFile("media/xxx.jpg")` |
| JSON 错误 | 检查是否有中文引号 `""` |
| 渲染慢 | 压缩图片到 < 5MB |
| 字幕看不清 | 增加背景遮罩透明度 |
| 切换生硬 | 调整淡入淡出时长 |
